Waarom noem je de eerste Html-pagina; index.html?

index.html is een HTML-bestand dat dient als startpagina voor een website.

Het is vaak het eerste bestand dat bezoekers van een website te zien krijgen.

Normaal gesproken wordt index.html automatisch geopend wanneer bezoekers het domein betreden zonder een specifiek bestand op te geven.

Met welke Html versie werken we nu?

HTML5 is de nieuwste versie van HTML

Met welke css versie werken we nu?

CSS3 is een vernieuwde versie van het oude bekende CSS(Cascading StyleSheet).

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (edge,chrome,firefox,Opera)

De grootte van de tekst kan nét iets verschillen.

Bij het gebruik van “custom fonts” (die niet standaard op je computer staan) kan het ook voor komen dat ze in sommige browsers een beetje pixelig/rafelig worden.

Er zijn elementen die worden gestyled door je browser. En natuurlijk styled elke browser dit anders.
